# CScan

CScan is Next Generation Web Security Scanner with Futuristic Architecture and UI.

可以查看[竞品分析](http://note.youdao.com/noteshare?id=991ed4f0febff08daf5f53c70d3f09ef)，本仓库目录划分如下：

```yaml
- docs 项目文档
    - 竞品分析
    - 项目规划
    - 进度追踪

- scanner 扫描器相关代码
    - python Python 相关代码
        - poc POC 相关代码
        - general 通用扫描器相关代码
    - go Go 相关代码

- scheduler 调度器
    - docker 扫描单元的镜像封装

- terminal 终端设备
    - ui 设备控制界面
    - server 设备应用后台

- cloud 云端
    - ui 云端界面
    - 云端应用后台

- utils 辅助工具
    - build 构建工具
    - database 数据库设置与模型
```
